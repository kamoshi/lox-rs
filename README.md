# Loxy
Another Rust implementation of the Lox programming language

## AST interpreter
### Supported types
- Nil
- Boolean
- Number (float64)
- String (UTF8 string),

### Features
- [x] repl
  - [x] expressions
  - [x] statements   
- [x] print
- [x] variables
- [x] scope
- [ ] conditionals
- [ ] logic
- [ ] while
- [ ] for
